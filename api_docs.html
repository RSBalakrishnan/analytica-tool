🆔 Identity Management

POST /track/id

Generate a tracking identity for bulk outreach.

Request Body

{ "targets": ["https://site.com/pricing"] }

Response

{
  "trackingId": "V1StGXR8...",
  "pixelUrl": "http://3.110.134.234:3000/pixel/V1StG...",
  "redirectUrlBase": "http://3.110.134.234:3000/r/V1StG...?url="
}

📊 Analytics & Journeys

GET /analytics/tracking/:trackingId

Detailed breadcrumb timeline of the user's visit including per-page duration.

{
  "summary": {
    "timeSpentSeconds": 45,
    "timeline": [
      { "url": "/", "enteredAt": 1776422480506, "duration": 32 },
      { "url": "/pricing", "enteredAt": 1776422619926, "duration": 10 }
    ],
    "pagesVisited": [
      { "url": "/", "timeSpent": 37, "views": 2 }
    ]
  }
}
GET /analytics/email/:trackingId

Real-time status check for email opens and re-opens.

{
  "isOpened": true,
  "openCount": 2,
  "firstOpened": 1776422401574,
  "lastOpened": 1776422574095
}
GET /analytics/link/:trackingId

Chronological log of all generated links clicked by this specific ID.

{
  "clickCount": 2,
  "clicks": [
    { "timestamp": 1776422480240, "target_url": "https://site.com/pricing" }
  ]
}
GET /analytics/page?url=

Global aggregate performance metrics for a specific page URL.

{
  "url": "/pricing",
  "pageViews": 142,
  "uniqueVisitors": 89,
  "avgTimeSpentSeconds": 34
}

🧩 SDK Integration

Enable automatic tracking on any website with a single script tag.

<script 
  src="http://3.110.134.234:3000/sdk/analytica.js" 
  data-endpoint="http://3.110.134.234:3000"
></script>
  • Persists tid in localStorage automatically.
  • Fires PAGE_VIEW on initial load and SPA navigation.
  • Captures PULSE signals for active engagement heartbeats.
